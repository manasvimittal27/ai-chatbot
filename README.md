# 🤖 Terminal AI Chatbot

A simple, memory-capable AI assistant built with Python and OpenAI. This project allows you to have a continuous conversation with GPT-4o-mini directly from your terminal.

## ✨ Features
*   **Contextual Memory:** The bot remembers what you said earlier in the chat.
*   **Environment Security:** Keeps API keys safe using `.env` files.
*   **Lightweight:** Minimal code for maximum performance.

## 🛠️ Setup
1. Clone this repository.
2. Install the required libraries: `pip install -r requirements.txt`
3. Create a `.env` file and add your key: `OPENAI_API_KEY=your_key_here`
4. Run the app: `python3 chatbot.py`
